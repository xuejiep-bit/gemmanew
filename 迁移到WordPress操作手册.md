# GEMMANEW 网站迁移到 WordPress 操作手册
### 中英文双语 · Polylang 免费版 · 阿里云 + 宝塔

> 这份手册帮你把现在 `gemmaflor.com/gsjs/` 那套**静态导出**网站，
> 换成**直接用动态 WordPress 提供**，并用免费的 Polylang 插件统一管理中英文。
>
> 完成后的好处：
> - ✅ 改内容只要后台点一点，**不用再导出/上传静态文件**
> - ✅ Bing 点进去排版垮掉（CSS 没加载）的问题**自动消失**

---

## 阶段 0：开始前必看的 3 个前提

### 0-1. 备案（你服务器在大陆，必须做）⚠️
- `gemmaflor.com` 必须有 **ICP 备案**，否则在大陆阿里云服务器上打不开。
- 公司中文站应该已有备案 → 去 **阿里云控制台 →「备案」→「新增网站/新增域名」**，
  把 `gemmaflor.com` 加到现有备案里（比从零备案快）。
- 这是走流程，可能几天到一两周。**建议今天就先提交**，审核期间同时做下面的后台配置。

### 0-2. 先备份（任何操作前）
在宝塔里给网站和数据库都备一份：
- 宝塔 →「网站」→ 你的站点 →「备份」→ 备份网站文件
- 宝塔 →「数据库」→ 找到 WordPress 的库 →「备份」
- 出任何问题都能还原，安心操作。

### 0-3. 域名方案（免费版限制，先知道）
Polylang 免费版**不支持"中文一个域名、英文另一个域名"**。
免费做法 = **一个主域名 + 子目录**：
- `gemmaflor.com/zh/` → 中文
- `gemmaflor.com/en/` → 英文

老域名 `gemmanew.cn` 之后做个 301 跳转过来即可。

## 阶段 1：装 Polylang 并配置（WordPress 后台）

### 1-1. 安装插件
1. 浏览器登录 WordPress 后台（一般是 `你的域名/wp-admin`）
2. 左侧菜单 →「插件 Plugins」→「安装插件 Add New」
3. 右上搜索框输入 **Polylang**
4. 找到 **Polylang**（作者 WP SYNTEX）→ 点「现在安装 Install」→ 装完点「启用 Activate」

### 1-2. 添加两种语言
启用后会自动进入设置向导（或左侧菜单「Languages 语言」）：
1. 添加第一种语言：**中文（简体）/ Chinese (Simplified)**，代码 `zh-CN`
2. 添加第二种语言：**English**，代码 `en` 或 `en-US`
3. 把**中文设为默认语言**（Default）

### 1-3. 给现有内容打语言标签
向导会问"已有的页面/文章算哪种语言" → 先**全部归为中文**。
（稍后再把英文页一个个改成英文，见 1-4）

### 1-4. 把英文页改成 English，并和中文页配对
对每一个英文页面：
1. 「页面 Pages」列表 → 打开那个英文页 → 进入编辑界面
2. 右侧找到 **「Languages 语言」框**，把语言选成 **English**
3. 在同一个框里，有个"关联/翻译"的栏目，点 **+** 选择它对应的**中文页**
   - 例：英文 `PVC Homogeneous Sheet Flooring` ↔ 中文 `PVC 同质透心卷材`
4. 更新 Update 保存

> 配对的意义：访客在中文页点"切换英文"，会**精准跳到对应那一页**，
> 而不是傻乎乎跳回首页。每个页面都要配一次。

### 1-5. 加"语言切换"按钮到导航栏
1. 左侧「外观 Appearance」→「菜单 Menus」
2. 左边可添加项里勾选 **「语言切换 Language switcher」** → 添加到菜单 → 保存
3. 这样导航栏就有了自动的 CH/EN 切换，替换掉你现在写死的那个 `CH/EN` 链接

### 1-6. 设置 URL 形式
1. 「Languages 语言」→「Settings 设置」→ 找「URL modifications」
2. 选 **"The language is set from the directory name"（用子目录）**
   → 得到 `gemmaflor.com/en/`、`gemmaflor.com/zh/` 这种地址
3. 保存

## 阶段 2：宝塔 / 阿里云 配置（最容易踩坑，务必做）

### 2-1. 伪静态（不做的话 /en/ 这种地址会全部 404）⚠️
1. 宝塔面板 →「网站」→ 点你的站点
2. 左侧「伪静态」→ 模板选 **WordPress** → 保存
3. 回 WP 后台 →「设置 Settings」→「固定链接 Permalinks」
   → 选「文章名 Post name」→ **点一次保存**（很重要，刷新规则）

### 2-2. 绑定域名（让 gemmaflor.com 直接走 WordPress）
1. **阿里云 DNS 解析**：阿里云控制台 →「云解析 DNS」→ `gemmaflor.com`
   → 添加 **A 记录**，记录值 = 你这台服务器的**公网 IP**
2. **宝塔绑定**：宝塔 →「网站」→ 站点设置 →「域名管理」
   → 把 `gemmaflor.com` 加进去
3. 目的：访问 `gemmaflor.com` 直接打开动态 WordPress，
   **不再走 `/gsjs/` 那堆静态文件**

### 2-3. 配 HTTPS（免费证书）
1. 宝塔 →「网站」→ 站点设置 →「SSL」
2. 选 **Let's Encrypt** → 申请（免费）
3. 申请成功后打开 **「强制 HTTPS」**

## 阶段 3：验收检查

逐项确认：
- [ ] `gemmaflor.com`（不带 /gsjs/）能正常打开，**排版正常**
- [ ] 点导航栏的 CH/EN，能在中英文之间正确切换，且**停在对应的那一页**
- [ ] 直接复制粘贴某个内页地址到新标签打开，**样式不垮**（这就是修好了原来的毛病）
- [ ] 后台随便改一个字 → 保存 → 前台刷新，**马上生效**（不用再导出）

---

## 阶段 4：收尾（可选，备案通过后再做）

- 把老域名 `gemmanew.cn` 做 **301 跳转**到 `gemmaflor.com`（保留 SEO 权重）
- 原 `/gsjs/` 静态文件留作备份，确认新站稳定后可清理
- 在「设置 → 阅读」里确认搜索引擎可以正常收录

---

## 卡住了怎么办？

把**卡在哪一步 + 截图**发我，我帮你看。常见坑：
- `/en/` 打不开 → 八成是 **2-1 伪静态**没配
- 域名打不开 / 提示未备案 → **阶段 0-1 备案**没好
- 切换语言跳回首页 → **1-4 页面配对**没做
